## padding

Espaços entre os elementos

- padding-top | padding-right | padding-bottom | padding-left
- values: `<length>` | `<percentage>`

```css
    /* shorhand */
    padding: 12px 16px 10px 4px;
    padding: 12px 16px 0;
    padding: 8px 16px;
    padding: 8px;
```

* Padding poderá causar diferença na largura de um elemento
